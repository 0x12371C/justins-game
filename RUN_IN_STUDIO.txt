Collab branch workflow:
- Create branch:
  git checkout -b feature/my-task
- Push branch:
  git push -u origin feature/my-task
- Open PR into main and merge after review.
- After merge, both sync:
  git checkout main
  git pull origin main

Fishing UI behavior (current):
- UI only appears when a fishing pole is equipped and activated.
- Tool is identified by:
  - Tool attribute IsFishingPole = true (preferred), or
  - Name containing fish + pole/rod.

Inventory behavior (current):
- Backpack CoreGui is forced on by src\client\CoreGuiBackpack.client.lua
- Players get a Fishing Pole from src\server\FishingInventory.server.lua
